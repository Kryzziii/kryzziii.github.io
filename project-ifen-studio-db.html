Project description

It is a local web application I developed during one of my work placements. It is used to manage IFEN's audiovisual productions.

Architecture

The application features a responsive, dynamic web interface constructed with semantic HTML5, custom CSS3, and JavaScript to ensure smooth client-side interactivity. Server-side processing is handled by robust PHP scripts that facilitate secure data transmission.

The architecture relies on a relational MySQL database, which is fully integrated into the UI, allowing users to perform comprehensive data operations directly through the interface.

Scope of use

Functionally, the platform provides a robust CRUD interface for the manual indexing of media assets. Users can populate the database with granular tags regarding effects, transitions, and sound design, enabling complex search queries across all asset types. Crucially, the system supports the manual association of featured individuals with specific video entries. This metadata is continuously monitored by an alert system that notifies administrators when data retention dates are reached, ensuring strict adherence to statutory archiving timelines.
